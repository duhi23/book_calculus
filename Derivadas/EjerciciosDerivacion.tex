\item
Halle las ecuaciones de las rectas tangente y normal a la gráfica de la ecuación dada en el punto indicado.
        \begin{enumerate}
        \item
        $8x^2-7y+3x = 0$, \ \ en $(1,1)$.
        \item
        $4x-y^2+8y = 3$, \ \ en $(-1 ,1 )$.
        \item
        $y=\frac{x-1}{x+1}$, \ \ en $(0 , -1 )$.
        \item
        $x=\frac{y}{y^2+1}$, \ \ en $(\frac{1}{2} ,1 )$.
        \end{enumerate}
\item
Halle las ecuaciones de las rectas tangente y normal a la gráfica de $f$ en el punto donde la recta tangente sea paralela a la recta $l$ dada.
        \begin{enumerate}
        \item
        $f(x) = \sqrt{25-x^2}$, \ \ $l:\ 3x+4y+5=0$.
        \item
        $f(x) = 2x^2-3x+2$, \ \ $l:\ 2x-y+8=0$.
        \item
        $f(x) = 1+(x+1)^2$, \ \ $l:\ y=2x-7$.
        \item
        $f(x) = \frac{x+1}{x-1}$, \ \ $l:\ 2y-4x=5$.
        \end{enumerate}
\item
Halle la ecuación de la o las rectas que sean tangentes a las gráficas de $f$ y $g$ al mismo tiempo.
        \begin{enumerate}
        \item
        $f(x) = x^2+2x+2$, \ \ $g(x) = \frac{1}{2}(x+1)(3-x)$.
        \item
       $f(x) = 2x^2+6x+1$, \ \ $g(x) = x^2$.\ \ Respuesta: $l_1:\ y = 2x-1$, \ \ $l_2:\ y = -14x-49$.
        \end{enumerate}
